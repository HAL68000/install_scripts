# Ubuntu Installation Requirement

This script **requires Ubuntu** to be installed in order to run correctly.

Please follow the steps below carefully.

---

## Installation Steps

1. **Copy the required commands**  
   Click on **"COPIA COMANDI"** to copy all the necessary commands.

2. **Open the terminal**  
   Click on the **"APRI TERMINALE"** button to open a terminal window.

3. **Paste and run the commands**  
   Paste the copied commands into the terminal and execute them.

4. **Wait for installation to complete**  
   Allow the installation process to finish completely. This may take several minutes.

5. **Return to the previous screen**  
   Once the installation is complete, return to the previous screen to continue.

---

## Notes

- Make sure you have sufficient permissions to install software on your system.
- An active internet connection may be required during installation.
